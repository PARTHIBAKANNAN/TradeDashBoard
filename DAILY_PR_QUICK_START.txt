🚀 QUICK SETUP (Just 3 Steps!):

STEP 1: Create GitHub Token
  → Go to: https://github.com/settings/tokens/new
  → Generate new token (classic)
  → Name: "TradeDashBoard Daily PR Bot"
  → Select: ✓repo ✓workflow ✓read:user
  → Copy token (save it!)

STEP 2: Add Secrets to GitHub
  → Go to: https://github.com/PARTHIBAKANNAN/TradeDashBoard
  → Settings → Secrets and variables → Actions
  → Create 2 secrets:

    Secret 1 - GITHUB_TOKEN
    Value: [paste your token from Step 1]

    Secret 2 - JULES_API_KEY
    Value: AQ.XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
    (paste your actual Jules API key here)

STEP 3: Commit & Push
  → In terminal, run:
    git add .github/ scripts/ GITHUB_ACTIONS_SETUP.md
    git commit -m "chore: add daily PR automation with Jules"
    git push origin main
